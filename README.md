# ArtSCII - Browser-based Image to ASCII Converter

**ArtSCII** 是一个纯前端运行的图片转 ASCII 字符画工具。所有转换逻辑均在浏览器本地完成，无需上传服务器，安全、快速且隐私。

## ✨ 特性 (Features)

- 🔒 **隐私安全**：图片完全在本地处理，不会上传到任何服务器。
- ⚡ **实时转换**：基于 Canvas 像素分析，毫秒级生成结果。
- 🎨 **高度定制**：支持自定义输出宽度（20-200字符）。
- 💾 **便捷导出**：
  - **复制文本**：一键复制 ASCII 文本。
  - **保存图片**：将 ASCII 文本渲染为高清 PNG 图片下载，完美解决社交软件字体对齐问题。
- 📱 **响应式设计**：黑客风格 UI，适配桌面与移动端。

## 🚀 快速开始 (Quick Start)

### 方式一：直接打开
直接双击 `index.html` 在浏览器中运行（部分浏览器可能因 CORS 限制本地图片，推荐方式二）。

### 方式二：本地服务（推荐）
如果你安装了 Python：

```bash
# 启动本地服务
python -m http.server 8000

# 浏览器访问
http://localhost:8000
```

### 方式三：命令行工具
本项目也包含一个 Python CLI 版本：

```bash
# 安装依赖
pip install -r requirements.txt

# 运行转换
python ascii.py input.jpg -w 100
```

## 🛠️ 技术栈 (Tech Stack)

- **Core**: Vanilla JavaScript (ES6+)
- **UI**: CSS3 (Grid/Flexbox, CSS Variables)
- **Algorithm**: Luminance-based grayscale mapping
